# Python Calculator (Tkinter)

This is a simple calculator project developed in Python using Tkinter.  
It was created in an **educational context** as part of learning programming and GUI development.

## Features
- Basic operations: +, -, *, /
- Clear button
- Simple error handling (e.g., division by zero)
- User-friendly graphical interface

## Technologies
- Python
- Tkinter

## How to Run
1. Open the `calculadora.py` file in your IDE or Python terminal
2. Run the script (`python calculadora.py`)
3. The calculator GUI will appear
